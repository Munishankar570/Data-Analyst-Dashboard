# **Sales Profitability & Shipment Analysis Dashboard (Excel Project)**
## **1. Project Objective**
The objective of this project is to design an interactive Excel dashboard that provides a clear understanding of sales performance, profitability distribution, and shipment behavior across different business dimensions. The dashboard enables users to identify high-performing states, cities, products, and shipment modes using dynamic charts, slicers, and filtered views. It serves as a decision-support tool for evaluating markets, optimizing logistics, and improving product strategy.

## **2. Dataset Used**

**File:** `Sales Data Set for Dashboard.xlsx`

Key fields included in the dataset:

* Order ID, Order Date
* Customer ID, Customer Segment
* State, City
* Product Category, Product Name
* Ship Mode
* Sales, Profit, Quantity
* Discount, Shipping Cost

This dataset captures transaction-level sales and operational details used to build the Excel dashboard.


## **3. Key Questions (KIPs)**

The dashboard answers the following business questions:

1. Which shipment mode is used most frequently by customers?
2. Which states generate the highest profit?
3. Which cities contribute the most to overall profitability?
4. What are the top most profitable products in the dataset?
5. How do customer segments and product categories affect profit patterns?
6. How do different shipment modes influence operational performance?


## **4. Process Followed**

### **Data Cleaning:**

* Removed duplicates
* Standardized city/state names
* Ensured data types were consistent
* Checked for missing or invalid values

### **Data Preparation:**

* Created pivot tables for shipment mode, state profit, city profit, and product profitability
* Added calculated fields where needed
* Organized dataset for dashboard linking

### **Dashboard Development:**

* Built interactive visuals using Pivot Charts
* Added Slicers for:

  * Customer Segment
  * Product Category
  * Ship Mode
* Applied formatting, color-coding, and layout design for clarity
* Linked all slicers for synchronized filtering

## **6. Project Insights**

* **Regular Air** is the most commonly used shipment mode.
* A few key states and cities account for the majority of profit, indicating strong geographic concentration.
* High-margin products significantly influence overall profitability.
* Customer preferences differ by segment, category, and shipment method.
* Shipment patterns suggest opportunities for cost optimization and delivery planning.


## **7. Final Conclusion**

The Excel dashboard successfully transforms raw sales data into a meaningful, interactive analytical tool. It provides deep visibility into sales and profit trends, helping stakeholders identify the most profitable regions, products, and shipment behaviors. The dynamic slicers and visual summaries make the dashboard easy to use and highly valuable for decision-making across sales, logistics, and product teams.
